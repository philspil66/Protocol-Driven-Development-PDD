# Overview

This folder contains short orientation documents for the Protocol Driven Development (PDD) repository.

## Contents

- **[what-this-is-not.md](what-this-is-not.md)** — What this repo is *not*: framework, methodology, product, or universal rules. Sets expectations and boundaries.
- **[who-this-is-for.md](who-this-is-for.md)** — Intended audience: people responsible for software that must keep working, and who care about change, risk, and long-term consequences.

Start here if you are new to the repo. The main [README](../README.md) points to these as the first stop.
